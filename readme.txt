== Description ==
Adds an Event custom post type with Event Type taxonomy, meta fields, templates, UI-based filtering/search (with AJAX), RSVP, REST endpoints, email notifications, and WP-CLI commands.

== Installation ==
1) Upload folder to wp-content/plugins/eventsphere-manager
2) Activate plugin
3) Visit CPT archive: /events
4) Helper page is auto-created: /events-list ([event_list])
